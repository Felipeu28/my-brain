## Summary
A productized service model where AI tools (primarily Claude Code skills) are used to audit SMB business functions — Google Ads, SEO, website CRO — and deliver structured reports. The audits are positioned as high-value consulting outputs, enabled by Claude Code skills that run 190+ checks automatically. Multiple X creators published guides on this in the bookmark period, positioning it as a scalable way to land SMB clients.

## The Niche Playbook Pattern
Recurring across multiple creators: pick a single SMB niche (dentists, gyms, med spas) + AI-powered audit → low CAC + high conversion because the audit demonstrates value immediately. This is essentially Moil's own model (21 questions → instant market research output = free audit that converts to subscription).
