**What:** Entrepreneurship incubator/accelerator run by [[Buda EDC]] (under [[Jacquie Martinez]])
**Location:** Buda, TX

## Model

Cohort-based — groups of local entrepreneurs coached through curriculum by Andres Urrego + [[Joshua Edmond]] (joint contract as Incubator Strategists).

## Cadence

- 3 cohorts/year
- 3 BFF events aligned with: Texas Venture Fest (October), SO SX / South by Southwest (March), Entrepreneur Camp (June)
- Ongoing 1:1 sessions with entrepreneurs
- Flexible networking opportunities
- October–December: no active cohort, but recruitment + networking events continue

## Moil Integration

[[Buda EDC]] purchases [[Moil]] platform licenses → each cohort participant receives one license. This is a **B2G (business-to-government) recurring revenue** relationship. Entrepreneurs learn Moil during the incubator, creating loyal users who continue after graduation.
