## What is Connectex?

A **Next.js 16 SaaS platform for a technology advisory company targeting Austin SMBs.** It combines a public marketing site with a full internal CRM.

**Stack:** Next.js 16 (App Router, Turbopack), React, TypeScript, Tailwind CSS, Supabase, Google/Gmail/Gemini integrations, deployed on Vercel

**Two distinct surfaces:**
1. **Marketing site** — public routes: homepage, about, solutions, resources, contact, partners, marketplace — built from static components and data files
2. **CRM app** — protected under `/crm/*` — dashboard, contacts, pipeline, email campaigns, sequences, calendar, tickets

**Integrations:** Supabase (PostgreSQL + auth), Google/Gmail API, Google Gemini AI, cron jobs via API routes

This is a sophisticated build — essentially a full business operating system for an Austin tech advisory firm.
