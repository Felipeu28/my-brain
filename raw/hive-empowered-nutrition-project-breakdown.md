🚀 AI-Powered Website Development Guide for Empowered Nutrition with Inna
1. The Master Prompt Strategy
Instead of sending one massive prompt (which risks cut-offs and high token usage), break it into 6 modular prompts:
Project Definition &amp; Branding
Layout &amp; UX Wireframe
Visual Design &amp; Style Guide
Content Development (Section by Section)
Conversion &amp; Engagement Features
SEO, Performance &amp; Launch Optimization
Each prompt “feeds” the next. For example: Prompt 1 defines brand identity → Prompt 2 builds structure → Prompt 3 applies design → Prompt 4 generates copy → Prompt 5 layers in conversions → Prompt 6 optimizes.
2. Prompt Modules
Prompt 1 – Project Definition &amp; Branding
Goal: Define voice, market, and identity.
You are a senior brand strategist and web designer. We are creating a modern, high-converting website for "Empowered Nutrition with Inna". Branding kit: - Dark Green #183b10, Medium Green #256439, Light Green #53a107- Accents: Yellow #FCAF15, Red #CE0B42, Purple #680C5F, Pink #F9A7D4, Teal #29C5EF- Introduce Coral #E88074 as secondary accent. Define:1. Target audience (demographics, pain points, motivations).2. Core value proposition and differentiators.3. Brand voice and personality (tone, language style).4. Emotional response the site should trigger.5. Design keywords (e.g., clean, professional, warm, scientific).6. Competitor benchmarks in wellness/nutrition.Deliver a concise brand blueprint with personality traits, style, and audience insights.
Prompt 2 – Layout &amp; UX Wireframe
Goal: Map the page structure.
You are a UX architect. Using the brand blueprint, design a one-page website wireframe for Empowered Nutrition with Inna. Include sections:- Hero with video/photo + CTA- Animated symptom display- Introduction to Inna- “How We Help” with four pillars- Services grid (conditions treated)- My Story (video/text)- Testimonials- Educational resources/blog preview- Book consultation CTA with embedded chat- FooterDeliver:1. Section order (wireframe flow).2. Suggested layouts (columns, grids, carousels).3. Interaction design (animations, hover effects).4. Notes on user journey and CTA placement.
Prompt 3 – Visual Design &amp; Style Guide
Goal: Translate branding into UI.
You are a UI/visual designer. Based on the brand kit and UX wireframe, create a full style guide for the website. Include:1. Typography pairings (headings, body, accent).2. Color usage rules (primary, secondary, accents).3. Button and CTA styling (normal, hover, active).4. Iconography and illustration style.5. Image treatment (filters, framing, backgrounds).6. Animation guidelines (e.g., fade-in hero video, carousel speed).7. Accessibility considerations (contrast ratios, font sizes). Deliver a style system that developers can apply consistently.
Prompt 4 – Content Development
Goal: Generate optimized copy, section by section.
Break into sub-prompts for each section:
4a. Hero Section Copy
You are a copywriter specializing in healthcare websites. Write hero section copy with:- Headline (5–7 words, benefit-driven)- Subheadline (1 sentence, outcome-oriented)- CTA button text (action-driven).Tone: professional, empathetic, inspiring.
4b. My Story Section
Write a heartfelt "My Story" section for Inna. Include her daughter’s health journey, her motivation to start this practice, and her commitment to evidence-based nutrition. Length: 250–300 words.
4c. Testimonials
Generate 3 sample testimonials (short, medium, long) highlighting trust, results, and personal transformation. Tone: authentic and personal.
…and so on for each section (Services, How We Help, Blog Previews).
Prompt 5 – Conversion &amp; Engagement
Goal: Optimize for leads and appointments.
You are a CRO (conversion rate optimization) specialist. Design conversion flow for Empowered Nutrition with Inna. Include:1. Live chat script (welcome message, lead capture questions: name, email, phone, symptoms).2. CTA button placements and variations.3. Lead magnet idea and positioning.4. Sticky CTA or floating button logic.5. Funnel journey: visitor → engaged → booked consult → follow-up.
Prompt 6 – SEO, Performance &amp; Launch Optimization
Goal: Ensure discoverability and performance.
You are an SEO strategist and technical web consultant. Apply SEO and performance optimization to the Empowered Nutrition with Inna site. Deliver:1. Meta titles &amp; descriptions (5 versions).2. Suggested keywords (short-tail, long-tail, local).3. Blog content ideas (10 topics).4. Internal linking plan.5. Structured data schema (LocalBusiness, MedicalClinic).6. Mobile and speed optimization checklist.7. Accessibility compliance checklist.
3. Development &amp; Build Workflow
Run Prompt 1 → Brand Blueprint
Run Prompt 2 → UX Wireframe
Run Prompt 3 → Style Guide
Run Prompt 4 (sub-prompts) → Copy for each section
Run Prompt 5 → Conversion flow &amp; chat scripts
Run Prompt 6 → SEO &amp; performance plan
Assemble all outputs into one project document.
Implement in chosen platform (WordPress + Elementor / Webflow / React) applying outputs directly.
